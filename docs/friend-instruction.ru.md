# Инструкция для пользователя: режимы интернета для выбранного устройства

На роутере настроены отдельные режимы подключения для одного выбранного устройства.

Остальные устройства остаются на обычном интернет-подключении через Default policy.

## Основной режим

Используйте:

**CLIENT-PROFILE-A**

Это основной режим.

## Резервные режимы

Если основной режим не работает, переключите выбранное устройство на:

**CLIENT-PROFILE-B**

или:

**CLIENT-PROFILE-C**

## Прямой режим

Чтобы отключить proxy/VLESS для выбранного устройства, верните его в Default policy.

После этого устройство будет использовать обычное SIM/WAN-подключение, как остальные устройства.

## Как проверить текущий IP

Откройте на выбранном устройстве:

```text
https://api.ipify.org
```

Если IP отличается от IP прямого SIM/WAN-подключения, значит выбранное устройство использует один из proxy-режимов.

## Что не трогать

Не меняйте и не удаляйте:

- `XRAY-PROFILE-A`
- `XRAY-PROFILE-B`
- `XRAY-PROFILE-C`
- `CLIENT-PROFILE-A`
- `CLIENT-PROFILE-B`
- `CLIENT-PROFILE-C`
- Default policy
- USB-накопитель
- OPKG / Entware / Xray settings

## Быстрый откат

Если интернет на выбранном устройстве перестал работать:

1. Откройте Web UI роутера.
2. Найдите выбранное устройство.
3. Переместите его обратно в Default policy.
4. Проверьте интернет на выбранном устройстве.
