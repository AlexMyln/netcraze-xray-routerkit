# Инструкция для пользователя: режимы интернета для TV

На роутере настроены отдельные режимы подключения для устройства **TV**.

Остальные устройства остаются на обычном/default интернет-подключении.

## Основной режим

Используйте:

**TV-VERTPATH**

Это основной режим.

## Резервные режимы

Если основной режим не работает, переключите TV на:

**TV-WHITE5**

или:

**TV-FROST2**

## Прямой режим

Чтобы отключить proxy/VLESS для TV, верните TV в default policy.

После этого TV будет использовать обычное SIM/WAN-подключение, как остальные устройства.

## Как проверить текущий IP

Откройте на TV:

```text
https://api.ipify.org
```

Если IP отличается от direct SIM/WAN IP, значит TV использует один из proxy-режимов.

## Что не трогать

Не меняйте и не удаляйте:

- `XRAY-VERTPATH`
- `XRAY-WHITE5`
- `XRAY-FROST2`
- `TV-VERTPATH`
- `TV-WHITE5`
- `TV-FROST2`
- default policy
- USB-накопитель
- OPKG / Entware / Xray settings

## Быстрый откат

Если интернет на TV перестал работать:

1. Откройте Web UI роутера.
2. Найдите устройство **TV**.
3. Переместите его обратно в default policy.
4. Проверьте интернет на TV.
